## 技术洞察
- LakeFill 的核心贡献：让 LLM 知道"什么时候该严格，什么时候可以放松"
- 三种置信度评分中，NLN（Length-Normalized Logprob）最适合 API 场景
- Checklist-based annotation 是构造 reranker 训练数据的高效方案
- 填表项目的最大短板：缺少置信度评分层 → 导致"填了不知道对不对"
