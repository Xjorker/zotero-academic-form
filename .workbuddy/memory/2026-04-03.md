# 2026-04-03 工作日志

## Bug 修复：user_context 未传递导致 Agent 用假名查 DBLP

**问题**：Agent 填表时 `fetch_academic_data` 工具报错 `未从 DBLP 找到 'Zhang San' 的相关论文`

**根因**：`academicForm.ts` 第 439 行声明了 `const userContext = author.name`，但第 449 行调用 `callAgentFill()` 时**漏传了 `user_context: userContext`**，导致后端收到的 `user_context` 为空字符串。LLM 在 system prompt 中看到"用户上下文: 无"，就自己编造了 "Zhang San" 作为作者名去查询 DBLP。

**修复**：在 `academicForm.ts` 的 `callAgentFill` 调用中补上 `user_context: userContext` 字段

**文件**：`src/modules/academicForm.ts`
